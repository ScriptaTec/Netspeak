# Netspeak

# Rodar o projeto 

Clonar o projeto

```bash
git clone git@github.com:ScriptaTec/Netspeak.git
```
Copiar o .env-docker para .env e adicionar as informações necessarias das variaveis de ambiente

```bash
cd src
```

```bash
cp .env-docker .env
```

Rodar o compose install dentro de src
```bash
composer install
```
